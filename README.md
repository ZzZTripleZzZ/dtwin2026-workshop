# DTwin 2026 Workshop Website

## Local preview
- From project root: `python3 -m http.server 8080`
- Open: `http://127.0.0.1:8080/index.html`

## GitHub Pages deployment (for collaborators)
1. Create a GitHub repository and upload this folder as the repository root.
2. In repository settings, enable **Pages** and select:
   - Source: `Deploy from a branch`
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
3. Save settings and wait for the Pages build.
4. Share the generated URL.

## Main content pages
- `index.html` (home + dates + registration notes)
- `cfp.html` (call for papers)
- `cfp-poster-demo.html` (posters and demos + award)
- `keynote.html` (keynotes)
- `panel.html` (panel details)
- `committee.html` (committees)
- `camera-ready.html` (submission and registration)
